💬 채팅 기능
전체 채팅

1:1 채팅

중요 공지 등록 및 모아보기. 공지 등록 시 상단 배너에 자동 고정
✉️ 메시지 기능
메시지 삭제/수정
내 메시지에 마우스 올리면 삭제/수정 버튼 표시. 수정 후 (수정됨) 표시
답장
메시지에 마우스 올리면 ↩️ 버튼. 원본 메시지 인용하여 답장
메시지 더보기
100자 이상 메시지는 자동으로 접힘. 더보기 클릭 시 전체 표시
연속 메시지 묶기
같은 사람이 연속으로 보내면 이름은 처음 한 번만, 시간은 마지막에만 표시
🖼️ 이미지 기능
이미지 붙여넣기
클립보드 이미지를 Ctrl+V로 바로 전송. 자동 압축 적용
드래그앤드롭
이미지 파일을 채팅창에 끌어다 놓으면 전송
이미지 확대 보기
이미지 클릭 시 전체화면으로 확대. 저장/복사/새 탭 열기 가능
